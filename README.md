SHACL_TQ
=============

Wrapper of [SHACL](https://github.com/TopQuadrant/shacl) project 
that works in Scala.

This project is used in [rdfshape](https://github.com/labra/rdfshape) 
 in order to be able to use also the TopQuadrant implementation 
 to validate RDF.
